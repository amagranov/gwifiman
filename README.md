# gwifiman
Реализует возможность автоматической смены пароля для гостевой WiFi сети на роутерах с прошивкой OpenWRT.

# Для чего использовать?
Часто курьеры, осуществляющие доставку "просят WiFi". Необходимость каждому рассказывать название сети, пароль помноженная на старческую параною относительно того курьер может "поделиться" твоим WiFi и что со временем полгорода будет "сидеть" в твоей домашней сети родила данный проект.

# Идея
Нужен механизм, который будет периодически менять пароль на гостевом WiFi и способ быстрого подключения для гостя.
Способом подключения был выбран QR код, отображаемый на экране собственного смартфона, который можно предъявить гостю.

# Как работает?
На роутере есть встроенный uHTTPd http сервер, который позволяет хостить на нем свой "сайт". Пользуясь этой возможностью разместим на нем страницу, которая будет отображать QR код для подключения к гостевой WiFi сети.
Cron будет запускать shell скрипт, который будет создавать новый пароль для этого WiFi, изменять его настройки на роутере и заменять javascript, отображающий QR код на любом устройсте, которое уже подключено к домашней сети.

# Установка
Если еще нет настроенной гостевой WiFi сети, настраиваем ее по одному из гайдов. Например:
https://openwrt.org/docs/guide-user/network/wifi/guestwifi/configuration_webinterface
https://lsetc.ru/gostevoj-wi-fi-na-openwrt-cherez-luci/
https://developersu.blogspot.com/2015/08/wifi-openwrt-luci.html

Затем подключаемся к роутеру по SSH.
Качаем скрипт install.sh, например командой wget -O install.sh https://github.com/amagranov/gwifiman/blob/main/install.sh
Даем разрешения на запуск chmod +x install.sh и запускаем ./install.sh
В процессе установки скрипт запрашивает какую сеть из настроенных на роутере использовать в качестве гостевой, какое шифрование у этой сети (обычно WPA), является ли сеть скрытой или отображается в списке сетей (я использую скрытую) и через какой интервал времени производить смену пароля.
После окончания установки для отображения QR кода нужно открыть в браузере на устройстве, подключенном к домашней сети ссылку http://{IP адрес роутера}/guest.html
По ссылке отображается QR код и обратный отсчет со временем действия текущего пароля.
После первой установки или перезагрузки роутера обратный отсчет может не соответствовать реальному времени действия пароля, т.к. смена производится по расписанию через cron job.
